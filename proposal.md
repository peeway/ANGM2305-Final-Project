<https://github.com/peeway/ANGM2305-Final-Project.git>

## Description
Run a game of Tetris using pygame. It pertains to creating entertainment using digital art

## Features
- Movement/rotation of blocks
	- use position code to and keybind inputs to create the movement/rotation.
- Line clear
	- use for loops to check if row(s) is clear and move all blocks down after.
- Score
	- create function that adds to score display depending on the value given.
- Game Over
    - use boolean data to see if blocks reach certain point or cannot be placed.
- Restart
    - create a function to clear the grid by setting it to 0

## Challenges
- Creating a grid
- Keybinds to move/rotate blocks
- How to restart the game
- How to implement sounds

## Outcomes
Ideal Outcome:
- python will open a pygame tab, allowing the user to move/rotate the tetris blocks and clear lines while a score is being displayed. if user blocks out, game will end but can restart.

Minimal Viable Outcome:
- python will open a pygame tab, allowing the user to move/rotate the tetris blocks and clear lines. if user blocks out, game will end.
