## Description
My final project is a recreation of Tetris, coded into python. Its main features are the ability to move and rotate blocks or tetrominoes using the arrow keys, 
however there is not game over feature so the cannot end unless the application is closed. 
The different files in my project are the essentials like the requirements to use the final project, the original proposal, and the README.md.
Design improvements I really would want to implement later is the Game Over and Restart feature for better quality of life improvements to the project.
I really enjoyed this project, implementing what was learned across the semester and understanding how to read the code. An aspect I also enjoyed was 
outsourcing help from other talented programmers incase I needed to understand a new concept.

A challenge I faced in this project was learning how to create the blocks. I looked for help on youtube, which led me to discover the inheritance classes. I think to improve my project better,
I focus on better time mangagement.
