## Description
 Animation application to create a short animation and it is revelvant as it pertains to animation media.

## Features
- Mouse Draw
	- Use a mouse interaction code to detect when mouse is click/held to leave a mark/drawing.
- Move between frames
	- Arrows are displayed at the bottom of the tab. When mouse interacts with arrows, the tab will seemlessly move back to the previous frame to be edited.
- Play Completed Animation 
	- "Finish" button will be displayed. When interacted with, program will compile all frames and play the animation by opening a file or within the tab.

## Challenges
- Mouse Interaction in tab
- Be able for the mouse to draw in the tab
- How to create and save frames drawn
- How to see previous frames and be colored to distinct them from current frame
- How to edit previous frames

## Outcomes
Ideal Outcome:
- python will open a tab, allowing for the user to draw and see onion skinning or go back to previous frames. When finished, the user will be able to see the completed animation.

Minimal Viable Outcome:
- python will open the tab, allowing the user to draw in different frames. When finished, the user will be able to see the completed animation.
